# Dataset Access

This project uses the **OASIS Longitudinal MRI Dataset**.

The dataset cannot be redistributed through this repository.

To obtain the dataset:

1. Visit the OASIS project website
   https://www.oasis-brains.org/

2. Register for dataset access.

3. Download the dataset:

```
oasis_longitudinal.xlsx
```

4. Place the file inside the project directory before running the notebook.

```
RiskLens-AD/
└── data/
    └── oasis_longitudinal.xlsx
```
